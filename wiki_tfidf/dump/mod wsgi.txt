mod_wsgi is an Apache HTTP Server module  by Graham Dumpleton that provides a WSGI compliant interface for hosting Python 2.3+ based web applications under Apache. As of version 3.4, mod_wsgi supports Python 2.6 and 3.2.

It is an alternative to mod_python, CGI, and FastCGI solutions for Python-web integration.
